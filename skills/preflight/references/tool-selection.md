# ツール選択マトリクス

## 動詞 → ツール マッピング

| タスクの動詞・文脈 | 主ツール | 補助ツール |
|-------------------|---------|-----------|
| 「探す」「検索」「どこにある」「一覧」 | Grep, Glob | Read |
| 「読む」「見る」「確認」「内容を見て」 | Read | Glob |
| 「実行」「ビルド」「テスト」「インストール」 | Bash | — |
| 「編集」「修正」「変更」「書き換え」 | Edit | Read（必ず先に読む） |
| 「作成」「生成」「新しく」 | Write | — |
| 「調べる」「調査」「最新」「ドキュメント」「仕様」 | WebSearch | — |
| 「分析」「まとめ」「理解」 | Read + Grep | Bash |
| 「リファクタ」「整理」 | Read + Edit | Bash（テスト確認） |
| 「削除」「消す」「取り除く」 | Edit または Bash | Read（確認後） |
| 複数領域・複雑・並列処理 | Agent | — |

## 複雑度によるエスカレーション

- **単一ファイル・既知パス** → Read のみ
- **パターンで複数ファイル** → Grep + Glob
- **コードベースの構造が不明** → まず Glob、次に Read
- **外部知識・最新情報が必要** → WebSearch を先に
- **副作用がある操作** → 必ず Read/確認 → その後 Write/Edit/Bash

## 避けるべき組み合わせ

- **WebSearch + Write（ローカル確認なし）**: 外部情報を確認せずにローカルファイルへ書き込む
- **Edit（Read なし）**: 読まずにファイルを編集する
- **Bash（rm/削除系）（確認なし）**: ユーザー確認なしに削除コマンドを実行する

## ツール説明（簡易）

| ツール | 説明 |
|--------|------|
| Read | 特定ファイルの内容を読む |
| Glob | ファイル名パターンでファイルを探す |
| Grep | ファイル内の文字列・正規表現を検索 |
| Edit | 既存ファイルの一部を書き換える |
| Write | ファイルを新規作成または全体を書き換える |
| Bash | シェルコマンドを実行する |
| WebSearch | ウェブ検索（外部情報・最新情報） |
| Agent | サブエージェントに複雑なタスクを委譲 |
